## Created Today
```dataview
TABLE string(split(string(file.ctime), " - ")[0]) as "Created", file.mtime as "Last Modified"
WHERE contains(file.path, "2025-09-24")
  OR (file.cday = date("2025-09-24") AND !regexmatch("^\d{4}-\d{2}-\d{2}", file.name))
SORT file.name ASC
```

## Last Modified Today
```dataview
TABLE string(split(string(file.mtime), " - ")[0]) as "Last Modified"
WHERE !contains(file.path, "2025-09-24")
  AND !contains(file.path, "2025-09-24 Daily Note")
  AND file.mday = date("2025-09-24")
SORT file.mtime DESC
```
