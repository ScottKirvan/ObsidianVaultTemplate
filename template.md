Obsidian Template with my favorite plugins, settings, and themes installed.
## Obsidian Settings
- Appearance:  
	- GitHubDHC

- Community Plugins:
	- Folder Note
	- Homepage
	- Obsidian Git (you'll want to adjust the git repo path per project - usually it's the current working directory, aka `.`)
	- Settings-search
	- Plugin Update Tracker

- Files & Links
	- New link format -> Relative path to file
	- Use Wikilinks off
	- detect all file extensions
	- New attachments -> In notes/media

- Editor
	- Properties in document -> Hidden
	- Strict Line Breaks -> enabled
